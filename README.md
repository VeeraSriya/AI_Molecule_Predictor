# AI Molecule Predictor

## Overview

This project uses machine learning to predict aqueous solubility (logS) from molecular descriptors.

## Dataset

* Delaney ESOL Dataset
* 1144 molecules

## Descriptors Used

* MolLogP
* MolWt
* NumRotatableBonds
* AromaticProportion

## Machine Learning Model

Random Forest Regressor

## Performance

* R² Score: 0.8875
* RMSE: 0.7000

## Generated Outputs

* Feature_Importance.csv
* Prediction_Results.csv
* Model_Performance.csv
* Correlation_Heatmap.png
* Predicted_vs_Actual.png

## Key Finding

MolLogP was the most important descriptor for predicting solubility.
